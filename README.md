## Tesla Clone

It's a tesla clone webapplication in a reactJs x Redux

### Installation

You will need Nodejs and Yarn,<br>
Clone the repo: 
```
git clone https://github.com/Kaoxlaw/ReactJs-teslaClone.git
```
Then
```
cd ReactJs-teslaClone
yarn install
```
Run the webApp
```
yarn start
```

Enjoy!
